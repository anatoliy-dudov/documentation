# Ресурсы: обзор раздела

Рабочие инструменты, которые не привязаны к конкретному продукту
(TaskFlow или PayFlow), а используются в любом проекте технической
документации.

## Материалы в этом разделе

- [Глоссарий](glossary.md) — термины из API-документации, баз знаний,
  ITIL-процессов и стандартов ГОСТ, использованные в этом портфолио.
- [Чек-листы](checklists.md) — чек-листы ревью для API-документации,
  статей базы знаний, закрытия инцидента и release notes.
- [Дальнейшее чтение](further-reading.md) — стандарты, стилевые гайды и
  инструменты, на которые я опираюсь в работе.

## Зачем отдельный раздел «Ресурсы»

Глоссарий и чек-листы — не финальные артефакты вроде статьи базы знаний
или референса API, а инфраструктура, которая делает остальные материалы
согласованными между собой. Я выделяю её в отдельный раздел, а не
рассеиваю по конкретным примерам, по той же причине, по которой в коде
общие утилиты выносят в отдельный модуль: одно место для обновления,
и понятно, где искать.
